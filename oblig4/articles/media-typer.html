Media typer

Hva er medie typer?

I CSS er det mulig å bruke forskjellige stil avhengig av enheten. Disse enhetene er medie typer. Hvilken stil skal brukes med hvilken medietype kan spesifiseres direkte på HTML filen via media attributet eller via CSS regelen @media.

Det finnes flere media typer, men bare 3 av dem er aktuelle i dag takket være responsiv design;

  • screen - Standarden. Brukes til dataskjermer, nettbrett, smarttelefoner etc.
  • print - Brukes til skrivere/printere
  • speech - Brukes til skjermlesere som "leser" siden høyt
