CSS layouts

Website layouts

Lenge, lenge, lenge siden, i de gamle dagene... måtte man bruke floats for nettsidens layout...

float er en posisjonerings property som flyter en element til høyre eller venstre siden av foreldre container. Den ble brukt, og er fortsatt brukt, til å lage enkle nettside layouts.

For enkle nettsider funker det fint. Men når nettsiden begynner å bli større og mere komplisert så blir det vanskeligere å vedlikeholde, for da må man ta hensyn til flere ting samtidig om man ønskerå ha et spesifikt layout eller et responsivt nettside (f.ex. display, padding, width etc.).

Nå har jobben blitt gjort lettere for oss med CSS layout methodene flexbox og grid.

CSS Flexbox

Flexbox er en layout metode i CSS som kan benyttes med display: flex. Den er som navnet tilsier en flexibel box. Den lar oss blant annet;

  • Definere hvor barn-elementene flyter inni boksen (justify-content)
  • Wrappe barn-elementene til neste rad hvis det ikke er plass (flex-wrap)
  • Bestemme rettning som barn-elementene flyter mot (flex-direction)

På andre ord er Flexbox en ett dimensional layout metode, fordi den fungerer bare mot en rad.

CSS Grid

Grid er også en layout metode i CSS. Den kan benyttes med display: grid. Som navnet indikerer, definerer dette en grid container. En grid er som en tabell, og den har celler (barn-elementer).

I motsetning til Flexbox som er en dimensional (fungerer mot en rad), er Grid to dimensional siden den fungerer mot den hele definerte arealen (rad og kolonne) av containeret.

På grunn av dette kan vi blant annet;

  • Bestemme antall rader og kolonner, og størrelsen av cellene i containeren (grid-template-rows og grid-template-columns)
  • Eksplisitt plassere et barn-element på den spesifiserte cellen i containeren (grid-column-start og grid-column-end)

Siden vi kan bare plukke og plassere elementer hvor på griden vi ønsker, gir dette oss kontroll til å lage mange kompliserte layouts uten å stresse over manuell flyting.
