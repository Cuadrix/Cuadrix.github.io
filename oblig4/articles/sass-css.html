Sass CSS

Dynamisk kode

Sass (Syntactically Awesome Style Sheets) er et CSS preprocessor som legger til syntaktisk sukker som lar oss skrive dynamisk CSS kode. Den gir oss mulighet til blant annet;

  • Definere globale variabler som kan refereres til fra andre deler av koden
  • Nesting av elementer for ryddighet
  • Dele opp en stor fil til mindre deler

Resten av dens evne finnes på sass-lang.com

Sass vs CSS eksempel

En ting som er irriterende med vanlig CSS er at man må veldig ofte skrive inn samme verdiene for mange regler.

Ved bruk av variabler i Sass kan dette unngås;

Nå hvis vi trenger å bytte farget, trenger vi kun å gjøre det en gang ved å endre verdien av $primary-color.

Med vanlig CSS hadde vi måttet å spesifisere farget for hver element som vi vil endre det på;

På grunn av den dynamiske faktoren som Sass tilbyr, kan det være mere nyttig å benytte det i store prosjekter.
