SEO Viktighet og Implementasjon

Hva er SEO?

SEO(Search Engine Optimization) er aktiviteter som utviklere gjør til å optimisere nettsider for Web Crawlere av forskjellige søkemotorer.

Det er altså aktiviteter som gjøres av utviklere med mål om at Web Crawlere får med seg nødvendig informasjon fra nettsiden som kan brukes til å indexere det(lagre til søkemotorens database) sånn at det blir rangert så høyt som mulig på søke resultater.

Derfor er semantiske HTML tagger viktige. Web Crawlere kan forstå dem bedre enn generiske <div> tagger som ikke har noe semantisk mening. Så lenge kode spesifikasjonen for semantiske tagger er følgt, tar Web Crawlere det for gitt at taggene er brukt til det de er ment å brukes for. Dette gir Web Crawleren en generell ide av nettsidens struktur og innhold som hjelper med indexing av nettsiden på søkemotorens databaser.

Hva slags aktiviteter som kan gjøres for å optimisere en nettside er avhengig av søkemotoren og dems rettning med oppdateringer. Uansett av dette er det noen generelle ting som vi kan alltid gjøre;

  1. Bruk semantiske tagger
  2. Bruk passende nøkkel ord i innholdet
  3. Bruk beskrivende side titler
  4. Bruk <meta> deskripsjon tag
  5. Fjerne alle døde lenker
  6. Optimiser hastigheten av nettstedet
  7. Ha med deskripsjoner for media (f.ex. alt attribut)
  8. Ha med en sitemap.xml på rooten av nettstedet

PageSpeed Insights

Problemet er på grunn av bruket av store .png bilde filer, så den anbefalte å bruke nye bildet formater som kan redusere størrelsen av bilder og dermed forbedre hastigheten av nettsiden.

Problemet ble løst ved å konvertere .png filene til .webp. Dette er en minimalisk optimalisering siden den påvirker ikke så mye SEO.

Andre optimaliseringer gjort på nettsiden

  • Brukt "SEO" nøkkelordet på <meta> deskripsjon og flere headings
  • Brukt beskrivende text innenfor alt attributet og <figcaption> taggen
  • Legget til sitemap.xml på rooten av nettstedet
