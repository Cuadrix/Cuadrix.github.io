SEO Viktighet og Implementasjon

Hva er SEO?

SEO(Search Engine Optimization) er aktiviteter som utviklere gjør til å optimisere nettsider for Web Crawlere av forskjellige søkemotorer.

Det er altså aktiviteter som gjøres av utviklere med mål om at Web Crawlere får med seg nødvendig informasjon av nettsiden som kan brukes til å indexere det(lagre til søkemotorens database) sånn at det blir rangert så høyt som mulig på søke resultater.

Derfor er semantiske HTML tagger viktige. Web Crawlere kan forstå dem bedre en generiske <div> tagger som ikke har noe semantisk mening. Så lenge kode spesifikasjonen for semantiske tagger er følgt, tar Web Crawlere det for gitt at taggene er brukt til det de er ment å brukes for. Dette gir Web Crawleren en generell ide av nettsidens struktur og innhold som hjelper med indexing av nettsiden på søkemotorens databaser.

Hva slags aktiviteter som kan gjøres for å optimisere en nettside er avhengig av søkemotoren og dems rettning med oppdateringer. Uansett av dette er det noen generelle ting som vi kan alltid gjøre; På denne artikkelen skal jeg bruke spesifikasjonen nedenfor til å optimisere dette nettstedet;

  • Bruk semantiske tagger
  • Bruk passende nøkkel ord i innholdet
  • Bruk beskrivende side titler
  • Bruk <meta> deskripsjon tag
  • Fjerne alle døde lenker
  • Innholdet skal være text for: [kode eksempler]
  • Ha med deskripsjoner for media (f.ex. alt attribut)
  • Ha med en sitemap.xml på rooten av nettstedet
